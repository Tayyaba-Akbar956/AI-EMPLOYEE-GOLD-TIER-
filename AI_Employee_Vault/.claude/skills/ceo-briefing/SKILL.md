# CEO Briefing Skill (Gold Tier Upgrade)

## Purpose

Extends Silver's `report-generator` skill with:
- Real financial data from Odoo (invoices, expenses, account balances)
- Social media activity across all 4 platforms (LinkedIn, Facebook, Instagram, Twitter)
- Subscription waste detection
- Proactive suggestions with pre-created approval files
- Graceful degradation when Odoo is unavailable

## Trigger

- **Scheduled**: Monday 7:00 AM via Windows Task Scheduler
- **On-demand**: `orchestrator.py --task briefing`

## Output Location

`AI_Employee_Vault/Briefings/YYYY-MM-DD_Monday_Briefing.md`

## Data Sources

### 1. Odoo Financial Data (via MCP)
- `get_invoices(date_from, date_to)` - Revenue this week
- `get_overdue_invoices()` - Outstanding payments
- `get_expenses(date_from, date_to)` - Expenses this week
- `get_account_balance(account_code)` - Current cash position

### 2. SQLite Activity Log
- Task completion counts by source (email, whatsapp, linkedin, filesystem)
- Task completion counts by category (urgent, normal)
- Average response time
- Failed tasks requiring attention

### 3. Social Media Activity
- LinkedIn posts this week (from Silver linkedin_watcher logs)
- Facebook posts this week (from Gold facebook_poster logs)
- Instagram posts this week (from Gold instagram_poster logs)
- Twitter posts this week (from Gold twitter_poster logs)

### 4. Subscription Waste Detection
- Recurring expenses from Odoo with no recent activity
- Unused services flagged for review

## Graceful Degradation

If Odoo is unavailable:
1. Use SQLite financial logs as fallback
2. Add warning banner: `⚠️ Odoo unavailable - financial data may be stale`
3. Continue with other sections (tasks, social media)
4. Create `SYSTEM_ALERT_ODOO_*.md` for human attention

## Proactive Suggestions Logic

### Invoice Follow-up
- Trigger: Invoice overdue > 7 days
- Action: Pre-create email approval file with follow-up template
- Template includes: invoice number, amount, days overdue, client contact

### Subscription Waste
- Trigger: Recurring expense with no activity in 30 days
- Action: Pre-create task approval file to review/cancel
- Requires human approval before cancellation

### Budget Alert
- Trigger: Monthly expenses > 80% of budget
- Action: Flag in briefing, no approval file needed (informational only)

## Integration with Orchestrator

When orchestrator receives `--task briefing`:
1. Load this SKILL.md
2. Trigger Claude Code with briefing generation task
3. Claude queries Odoo MCP for financial data
4. Claude queries SQLite for task/activity data
5. Claude reads social media logs from `/Logs/YYYY-MM-DD.json`
6. Claude generates briefing markdown
7. Claude creates proactive approval files if needed
8. Save to `/Briefings/YYYY-MM-DD_Monday_Briefing.md`
9. Log to SQLite + JSON
